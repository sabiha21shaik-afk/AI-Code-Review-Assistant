# AI Code Review Assistant 🚀

An AI-powered web application that automatically reviews source code, detects potential issues, and provides improvement suggestions.

## 📌 Project Overview

AI Code Review Assistant helps developers analyze their code using an intelligent review system.

Users can upload code files and receive:
- Code quality score
- AI-generated suggestions
- Review history
- File management
- Secure user authentication

The goal of this project is to reduce manual code review effort and help developers write cleaner, better code.

## ✨ Features

✅ User Registration and Login  
✅ JWT Authentication  
✅ Upload Source Code Files  
✅ AI Code Review System  
✅ Code Quality Score  
✅ Review Suggestions  
✅ Review History  
✅ Dashboard Interface  
✅ Modern React UI
